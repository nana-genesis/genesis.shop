# ✦ GENESIS | Alta Costura Eletrônica ✦

##  Projeto:

**GENESIS** é uma loja virtual conceitual que une **alta costura** com **componentes eletrônicos reciclados**. 

## Funcionalidades:

| Funcionalidade | Descrição |
|----------------|-----------|
|  **Carrinho de Compras** | Adicione/remova produtos, ajuste quantidades com persistência |
|  **Persistência Local** | Carrinho salvo no localStorage do navegador |
|  **Design Responsivo** | Adapta-se perfeitamente a Desktop, Tablet e Mobile |
|  **Efeitos Visuais Imersivos** | CRT scanlines, constelações, chuva de dados, olhos de Lain |
|  **Sistema de Notificações** | Feedback visual para todas as ações do usuário |
|  **Logging Estruturado** | Sistema de logs para debug e monitoramento |
|  **Rate Limiting** | Proteção contra spam de cliques |
|  **UI/UX Única** | Interface com identidade visual cyberpunk dark |
